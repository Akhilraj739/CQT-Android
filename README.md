**CQT** (Custom Quick Tiles) is a powerful Android utility that allows you to add fully customizable tiles to your Quick Settings panel. Whether you want to launch specific apps, trigger system actions, or automate root tasks, CQT gives you the control you need directly from your notification shade.

## ‚ú® Features

### üõ†Ô∏è Customization
- **Launch Apps**: Create tiles to launch any installed application or specific activity.
- **Open URLs**: Quick access to your favorite websites or web-based tools.
- **Custom Icons**: Choose from a built-in library of icons or use the default app icons.

### ‚ö° System & Root Actions
- **Power Menu**: Reboot, Reboot to Recovery, or show the Power Dialog.
- **App Management**: Force-stop (kill) current/specific apps or clear app data instantly.
- **System Toggles**: Toggle **Mobile Data**, **NFC**, and **ADB over Wi-Fi** with a single tap.
- **Utility Tools**: Flashlight toggle and High-speed Screenshot capture.

### üõ°Ô∏è Smart Execution Modes
- **Accessibility Service**: Uses Android's Accessibility API to perform UI actions (Lock Screen, Screenshot) safely and reliably.
- **Root Support**: Deep system integration for advanced toggles and power actions.
- **No-Root Fallback**: Works for standard tasks like opening apps/URLs without requiring special permissions.

## üöÄ Getting Started

1. **Download & Install**: Build the APK from source using Android Studio.
2. **Permissions**:
   - Enable **Accessibility Service** for UI actions.
   - Grant **Root Access** (optional) for advanced system toggles.
3. **Configure**: Open the CQT app to set up your custom tiles.
4. **Add to Panel**: Pull down your Quick Settings, tap the 'Edit' (pencil) icon, and drag your new CQT tiles into the active area.
